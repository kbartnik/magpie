# Disappearing Messages vs Documentation

*How should activists balance disappearing-message OpSec against the value of documenting state violence for accountability and legal action?*

Disappearing messages protect operational communications from later discovery. But activists who are targets of state violence have a countervailing interest in documentation: evidence for criminal complaints, civil litigation, accountability journalism, and historical record. The same device hygiene that protects organizers from prosecution also prevents documentation of police misconduct.

The two interests can coexist through compartmentalization: use disappearing messages for operational coordination; use a separate, documentation-oriented channel (or separate device) for recording incidents. But this requires explicit discipline about which channel serves which purpose, and the documentation channel remains a liability if discovered.

No clean resolution exists. The correct balance is context-specific: the value of operational security against a prosecutorial adversary must be weighed against the value of documentation for accountability processes — and those may be the same adversary.
