# Robert Nystrom

Game developer and author. Wrote [[Crafting Interpreters]] (2021), which builds jlox (tree-walking interpreter) and clox (bytecode VM) for the Lox language from scratch, covering Pratt parsing, resolver passes, and garbage collection.
