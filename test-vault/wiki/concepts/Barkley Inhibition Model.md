# Barkley Inhibition Model

Russell Barkley (1997): ADHD is fundamentally a disorder of **behavioral inhibition**, not attention. Inattention and working memory deficits are *downstream consequences* of impaired inhibition.

## The Cascade

1. **Impaired inhibition** — the stop signal is weaker; prepotent responses fire before deliberation occurs
2. → **Working memory fails** — can't hold a mental representation long enough to act on it
3. → **EF components fail** — flexibility, self-regulation, planning all depend on WM
4. → **Inattention appears** — but attention isn't the root cause

## Prospective vs Retrospective Working Memory

Barkley distinguishes:
- **Retrospective WM (verbal):** self-talk, rule-following, internalized speech — less impaired in ADHD
- **Prospective WM (nonverbal):** holding a mental image of a future state and working toward it — specifically impaired in ADHD

This explains why ADHD individuals often *know* the rules but fail to apply them: the prospective representation decays before it can influence action.

## Hierarchy of EF

Barkley treats inhibition as foundational — other EF components (WM updating, flexibility) depend on it. This contrasts with three-component models that treat EF dimensions as parallel and correlated.
